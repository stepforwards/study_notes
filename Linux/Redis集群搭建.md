# 单机版安装Redis

> 1.  将Redis的安装包通过各种工具上传到Linux上，具体使用什么工具，这里就不说了。
> 2.  解压Redis `tar -xvf redis-3.0.0.tar.gz `
> 3.  因为Redis是用C语言书写的，所以需要安装C语言开发环境，对Redis进行编译，安装命令 `yum -y install gcc-c++`
> 4. 使用make命令进行编译
> 5.安装 `make FRIFX=安装路径 install`
