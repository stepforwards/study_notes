反射常用的属性：

class ：定义类关键字

Class ： Class类，是类的描述符

Constructor：构造函数的描述符。`Class clazz = Person.class`

Method ：方法描述符 

``` stylus
Constructor ctor = clazz.getConstuctor(...);
ctor.setAccessible(true);	// 设置可访问性
ctor.newInstance(....);
```

Field
字段（成员变量）描述符。

``` stylus
Field  f = clazz.getDeclaredField("name");

```
Modifier
	修饰符，判断是否有含有指定的修饰符的。
	

``` stylus
Modifier.isStatic(m.getModifier());
```
